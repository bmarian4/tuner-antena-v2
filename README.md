# Tuner Antena V2

This repository hosts the source code for the tuner antenna project.

## Description
This project includes functionality for tuning antennas using websocket communication.

## Files
- `tuner_antena_v2_optimized.ino`: Main source code for the project.
- `config.h`: Configuration settings for the project.
- `websocket_handler.h`: Handles WebSocket connections.
- `auto_calibration.h`: Provides auto-calibration features.
- `logging_system.h`: Contains logging functionalities.
- `pid_tuning.h`: PID tuning methods.
- `ota_updater.h`: Handles OTA updates.
